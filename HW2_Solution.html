Preprocessing - Data Cleanning + Data Enhancing

  1. Convert the numeric features ‘Survived’ and ‘Pclass’ into factors features:
df$Survived = as.factor(df$Survived)
df$Pclass = as.factor(df$Pclass)
  1. Some features like ‘PassengerId’ and ‘Ticket’ represent unique values so they won’t help us to predict if the passenger survived or not - lets drop both columns to make our dataframe smaller and more informative:
df = df[,-c(1,9)]
  1. The ‘Name’ feature won’t helps us to predict if the passenger survived but we can extract the title and the lastname (later for family size feature) from each name and use the new feature ‘Title’ to help us get a better prediction:
n = dim(df)[1]
title = rep(NA,n)
#surname = rep(NA,n)
for (i in 1:n){
  #new_surname = strsplit(as.character(df$Name[i]),",",fixed=TRUE)[[1]][1]
  splited_name = strsplit(as.character(df$Name[i]),",",fixed=TRUE)[[1]][2]
  new_title = strsplit(splited_name,".",fixed=TRUE)[[1]][1]
  title[i] = trimws(new_title)
  #surname[i] = trimws(new_surname)
}

  1. Now we can remove the ‘Name’ column and replace it with our new ‘Title’ feature:
df = df[,-c(3)]
df$Title = as.factor(title)
  1. ‘Cabin’ feature have 147 levels and many NA values so this feature isn’t helping us much in the prediction process. instead of using the ‘Cabin’ feature (since some passengers have more then one cabin) we will convert it to Cabin_Count feature that will represent the number of cabins that the passenger stayed in:
n = dim(df)[1]
cabin_count = rep(0,n)
for(i in 1:n){
  if(!is.na(df$Cabin[i]))
  {
    temp = strsplit(as.character(df$Cabin[i])," ",fixed=TRUE)
    cabin_count[i] = length(temp[[1]])
  }
}

  1. Now we can remove the ‘Cabin’ column and replace it with our new Cabin_count feature:
df = df[,-c(8)]
df$Cabin_count = as.factor(cabin_count)
  1. Now we will add a family_size feature (based on SibSp and Parch) and discretize into 3 groups in order to get a better prediction:
df$Family_size_temp = df$SibSp + df$Parch + 1

df$Family_size[df$Family_size_temp == 1] = 'singleton'
df$Family_size[df$Family_size_temp < 5 & df$Family_size_temp > 1] = 'small'
df$Family_size[df$Family_size_temp > 4] = 'large'
# remove the temporary column, SibSp and Parch
df = df[,-c(5,6,11)]
df$Family_size = as.factor(df$Family_size)

Modeling The Data

  1. First we need to split our dataframe into train (75%) and test (25%) dataframes in order to evaluate our model accuracy:
indices = sample(1:nrow(df),nrow(df)*0.75)
train = df[indices,]
test = df[-indices,]
#remove the class column from the test set and save it for later usage
test_survived = test$Survived
test = test[,-c(1)]

First model - C5.0

  1. Let’s start with a basic C5.0 decision tree and try to predict the test set:
#install.packages("C50")
library(C50)
set.seed(123)
C50_model = C5.0(Survived ~., data=train)
c50_pred = predict(C50_model,test)
print("Tree mean:")
## [1] "Tree mean:"
mean(c50_pred==test_survived)
## [1] 0.7713004
  1. Visuallize our C5.0 tree:
plot(C50_model)

Second model - rpart decision tree

  1. Decision tree model with rpart - first we build the tree then we choose the best ‘CP’ based on the minimal ‘XError’ and then we train the model again with the best ‘CP’ value and compare the results:
#install.packages("rpart")
library(rpart)
set.seed(123)
rpart_model = rpart(Survived ~ .,data = train)
rpart_pred = predict(rpart_model, newdata = test, type = "class")
print("Tree mean before prune:")
## [1] "Tree mean before prune:"
first_mean = mean(rpart_pred==test_survived)
print(first_mean)
## [1] 0.7713004
bestcp = rpart_model$cptable[which.min(rpart_model$cptable[,"xerror"]),"CP"]
rpart_model_pruned = prune(rpart_model, cp = bestcp)
rpart_pruned_pred = predict(rpart_model_pruned, newdata = test, type = "class")
print("Tree mean after prune:")
## [1] "Tree mean after prune:"
second_mean = mean(rpart_pruned_pred==test_survived)
print(second_mean)
## [1] 0.7713004
#choose the best tree
if(second_mean > first_mean)
{
    rpart_model = rpart_model_pruned
}
